## Description
2022 <버미와 수리> NFT Project 서버 레포지토리입니다.

## Core Features
- 스마트 컨트랙트 호출하여, 사용자의 요청에 따른 NFT 발행
- 사용자의 경기 결과 예측 데이터를 DB에 저장
- 사용자의 예측 정보와, 실제 경기 결과 비교하여 포인트 부여

## Tech Stack
- DB : MySQL, TypeORM
- Server : Nest JS & Typescript
- Infrastructure : Docker & Docker-compose, AWS
